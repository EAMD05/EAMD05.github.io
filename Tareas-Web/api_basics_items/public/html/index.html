POST /items

Registra uno o varios items en el catálogo.

Parámetros:

  • id: ID numérico del item (requerido)
  • name: Nombre del item (requerido)
  • type: Tipo del item (requerido)
  • effect: Efecto del item (requerido)

Respuestas:

  • 201 Created: Item(s) agregado(s) correctamente
  • 400 Bad Request: Cuando faltan atributos o el ID ya existe

POST /users

Registra uno o varios usuarios con sus items.

Parámetros:

  • id: ID numérico del usuario (requerido)
  • name: Nombre del usuario (requerido)
  • email: Correo electrónico del usuario (requerido)
  • items: Array de items del usuario (requerido)

Respuestas:

  • 201 Created: Usuario(s) agregado(s) correctamente
  • 400 Bad Request: Cuando faltan atributos, el ID ya existe o algún item no existe en el catálogo

PUT /users/:id

Actualiza un usuario existente por su ID. Solo actualiza los campos proporcionados.

Parámetros:

  • id: ID del usuario a actualizar
  • name: Nuevo nombre del usuario (opcional)
  • email: Nuevo email del usuario (opcional)
  • items: Nueva lista de IDs de items del usuario (opcional)

Respuestas:

  • 200 OK: Usuario actualizado exitosamente
  • 400 Bad Request: Uno o más items no existen
  • 404 Not Found: Usuario no encontrado
