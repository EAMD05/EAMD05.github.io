GET /items

Obtiene todos los items del catálogo.

Respuestas:

  • 200 OK: Lista de items en formato JSON
  • 404 Not Found: Cuando no hay items disponibles

GET /items/:id

Obtiene un item específico por su ID.

Parámetros:

  • id: ID numérico del item

Respuestas:

  • 200 OK: Item en formato JSON
  • 404 Not Found: Cuando el item no existe
