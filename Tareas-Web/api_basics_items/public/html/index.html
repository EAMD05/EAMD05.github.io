GET /items

Obtiene todos los items del catálogo.

Respuestas:

  • 200 OK: Lista de items en formato JSON
  • 404 Not Found: Cuando no hay items disponibles

GET /items/:id

Obtiene un item específico por su ID.

Parámetros:

  • id: ID numérico del item

Respuestas:

  • 200 OK: Item en formato JSON
  • 404 Not Found: Cuando el item no existe

POST /items

Registra uno o varios items en el catálogo.

Parámetros:

  • id: ID numérico del item (requerido)
  • name: Nombre del item (requerido)
  • type: Tipo del item (requerido)
  • effect: Efecto del item (requerido)

Respuestas:

  • 201 Created: Item(s) agregado(s) correctamente
  • 400 Bad Request: Cuando faltan atributos o el ID ya existe
