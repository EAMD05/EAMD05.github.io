## Descripción
Coin Collector es un juego de plataformas donde controlas un personaje que debe recolectar monedas mientras se mueve por el escenario. El juego presenta animaciones fluidas y controles intuitivos.

## Instrucciones para Jugar

### Controles
- **Tecla W:** Mover hacia arriba
- **Tecla S:** Mover hacia abajo
- **Tecla A:** Mover hacia la izquierda
- **Tecla D:** Mover hacia la derecha
- **Botón Restart:** Reiniciar el juego
- **Botón Back:** Regresar a la página principal de proyectos

### Objetivo
El objetivo del juego es recolectar todas las monedas que aparecen en el escenario. El jugador debe moverse por el mapa y recoger cada una de las monedas disponibles.

### Reglas
1. El jugador puede moverse libremente por el escenario
2. Las monedas desaparecen al ser recolectadas
3. El juego termina cuando se recolectan todas las monedas
4. El jugador no puede salir de los límites del escenario

## Cómo Ejecutar el Juego
1. Abre el archivo `index.html` en tu navegador web
2. El juego comenzará automáticamente
3. Usa las teclas WASD para mover al personaje
4. Recolecta todas las monedas para completar el juego
5. Usa el botón "Restart" si deseas reiniciar el juego
6. Usa el botón "Back" para regresar a la página principal de proyectos

## Características

- Sistema de colisiones
